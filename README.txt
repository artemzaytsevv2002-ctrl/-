# Виртуальная лаборатория по биологии (полный шаблон)

**Готово к публикации на GitHub Pages.** Разделы 5–9 (ФГОС) и 10–11 (углубленный, ЕГЭ), страница поиска, каталог лабораторных, заглушки H5P.

## Публикация
1. Создайте новый репозиторий на GitHub (например, `biolab`).
2. Содержимое папки `biolab-site` из архива загрузите в репозиторий (в корень).
3. Включите GitHub Pages (Settings → Pages → Deploy from branch → `main` → `/`).
4. Откройте адрес из секции Pages — сайт готов.

## Структура
- `index.html` — главная.
- `5-9/` — страницы 5–9 классов, в каждом: темы, лабораторные, проверочная.
- `10-11/` — модули, тренажёр ЕГЭ.
- `labs/` — страницы лабораторных (вставляйте свои H5P/симуляции).
- `h5p/` — положите сюда HTML‑экспорты из Lumi.
- `quizzes/` — шаблон проверочной (Google Forms).
- `data/search-index.json` — индекс для простого поиска.
- `assets/` — стили и скрипты.

## Вставка H5P (Lumi)
Экспортируйте вашу активность как **HTML (Standalone)** и поместите файл в `/h5p`. На нужной странице лабораторной смените `src` в iframe.

## Правки навигации
Все ссылки относительные — сайт будет работать и в корне (`username.github.io`) и в подкаталоге (`username.github.io/repo`).

Удачи!
